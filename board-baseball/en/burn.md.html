In this burn/ directory are the SVG and PDF directories of files used in the creation of final play products.

When beginning a new release, the svg/ directory should be emptied of every SVG file and the pdf/ directory should be emptied of every PDF file (leaving only the README files) so as not to leave the impression the burn files reflect the intermediate state of the design SVG file.

The files in this directory are created during the build process documented here.
