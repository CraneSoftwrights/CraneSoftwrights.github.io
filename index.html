Crane Softwrights Ltd. Git Repository

To reach the corporate home page, please proceed to http://www.CraneSoftwrights.com.

If that doesn't work or doesn't interest you and you are back to this page, then one of three things has happened:

  • (likely) you are here browsing from the https://github.com/CraneSoftwrights/CraneSoftwrights.github.io GitHub repository or you found this address on Ken's LinkedIn profile.
  • (possible) there is a temporary glitch reaching http://CraneSoftwrights.com and you found this page instead; please try again later
  • (inevitable in a disaster scenario) the Crane website is no longer up and running due to full retirement of its entire staff and/or a complete shutdown of the company and the web site

As part of our disaster planning, we have made all of our free resources available on GitHub for the community to use:

  • books/ - free books
  • resources/ - free developer resources
  • articles/ - articles and essays
  • CraneSoftwrights - root Git account

But, really, this is just for disaster planning and the Crane web site is the best way to go through Crane's materials.
