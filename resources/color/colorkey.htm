Color Key for DSSSL Color Declarations: colorkey.htm

This is the color key corresponding to the DSSSL Color Declaration File (named colordef.dsl), the DSSSL Entity Color Declaration File (named colordef.ent) and the Entity Color Declaration File (named colordef.ent) created by distilling numerous web pages with various color definitions primarily used as HTML page creation guidelines. (The obliquely named "aliceblue" is probably supposed to be "iceblue", yet it showed up in many web resources; spelling mistakes are also preserved.)

Only the definitions for the color space and the colors you want need be included in your script; it is not necessary to include all the definitions contained therein.

Redundant names for a given set of values have not been removed, as you may have heard of any one of the names for the given set of values.

The examples page named colorex.htm illustrates how to use these files.

If anyone has yet more color definitions that could be included in this file, they are welcome to send them to info@CraneSoftwrights.com
