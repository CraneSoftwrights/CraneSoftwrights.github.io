Crane's UBL 2.2-Pre-award (csd02wd01pa03 20170525-2200z) Skeleton creation tool is a browser-based application used to create a skeleton XML instance of an OASIS UBL Pre-award test document created by the OASIS UBL Pre-award subcommittee.

We would appreciate any feedback you have, or suggestions for changes and improvements; please forward your comments to info@CraneSoftwrights.com.

This web documentation is in the process of being written. For now, please review the instructions on each edit page.

This environment is tested to work on Safari, Chrome and Firefox. It does not work with Internet Explorer 8 (it has not been tested in the lab with other versions of IE).
