The RSS (Rich Site Summary/RDF Site Summary) description of the training courses and upcoming public events from Crane Softwrights Ltd. is found here:  RSS XML 

In addition to Dublin Core components, this information includes components from the RSS Events Module.

Other items in the vocabulary are from this namespace:

  • <csl:course-id> - a unique identifier for each course offered; used to match a course description's meta data with a course delivery's meta data
  • <csl:syllabus rdf:resource="URL"> - a pointer to the syllabus for the course
  • <csl:length quantity="number" unit="day/hour"> - a prose description of the length
  • <csl:status> - the status of the course (interest only or registration open)
  • <csl:registration rdf:resource="URL"> - a pointer to registration information
  • <csl:pricing rdf:resource="URL"> - a pointer to pricing information
