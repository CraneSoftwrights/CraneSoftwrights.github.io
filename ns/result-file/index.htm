This namespace URI is reserved for the use of the result-file.xsl imported stylesheet and any stylesheets importing that stylesheet. result-file.xsl is used in partnership with an importing stylesheet to write content to an external file simultaneously supporting different vendor extensions to do so.

The importing stylesheet defines the content to be written, the file to which it is to be written, and the method by which it is written. The result-file.xsl imported stylesheet will try different vendor extensions in succession until one is found that works.

Parameters:

  • $filename (string) - required href for result tree serialization
  • $contents (result tree fragment) - required content for the file
  • $method (string) - optional method for serialization (default 'xml')
